# TanzilTube

TanzilTube is a powerful video streaming platform that provides users with access to a vast library of educational and inspirational videos. The platform allows users to watch, upload, and share videos seamlessly.

## Features
- Extensive video library
- User-friendly interface
- Upload and share your own videos
- Search and filter videos by category
- Create playlists
- Support for HD streaming and multiple formats

## Tech Stack
- **Frontend:** React.js, Bootstrap
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Cloud Storage:** AWS S3
- **Authentication:** JWT, OAuth

## Installation
1. **Clone the repository**  
   ```bash
   git clone https://github.com/kamelfidel/tanziltube.git
   cd tanziltube
   ```  
2. **Install dependencies**  
   - For the backend:  
   ```bash
   cd backend
   npm install
   ```  
   - For the frontend:  
   ```bash
   cd frontend
   npm install
   ```  
3. **Set up environment variables**  
   Create a `.env` file in the backend directory and add the necessary variables such as database URI, JWT secret, etc.
4. **Run the application**  
   - Start the backend server:  
   ```bash
   cd backend
   npm start
   ```  
   - Start the frontend development server:  
   ```bash
   cd frontend
   npm start
   ```

## Usage
Once the application is running, you can access the platform by navigating to `http://localhost:3000` in your web browser. You can create an account, log in, upload videos, and start exploring the video library!
